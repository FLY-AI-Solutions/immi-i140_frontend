AI Opportunity Map

Section-3 generation
Use AI to turn report findings into exhibit categories, evidence requests, and draft exhibit labels.
Evidence validation
AI can classify uploaded files as acceptable, weak, duplicative, or missing context.
Section-1 drafting
AI can convert the finalized evidence stack into a first-person self-petition introduction.
Section-2 elaboration
AI can build prong-by-prong argument blocks with exhibit references and caveats.
Coherency pass
AI can spot contradictions, unsupported statements, repeated language, and citation drift.
Final PDF assembly
AI can help normalize headings, exhibit map wording, and document ordering before DOCX/PDF export.

Current Prototype Scope

  • Section generators are rule-based placeholders for now.
  • Evidence validation is a dummy action with review prompts.
  • Users can manually edit every section and add unlimited evidence rows.
  • PDF/PNG/JPG file slots are present for future validation and export logic.
  • The page can run from report-seeded data or fallback sample data.
